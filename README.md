### Bab 1: The amazing world of TensorFlow

Bab ini memperkenalkan TensorFlow sebagai framework pembelajaran mesin yang kuat. Dibahas peran TensorFlow dalam tugas deep learning dan pentingnya perangkat keras khusus seperti GPU dan TPU untuk pemrosesan yang efisien. Bab ini juga memberikan dasar untuk memahami konsep dasar TensorFlow dan aplikasinya dalam pembelajaran mesin.

### Bab 2: TensorFlow 2

Bab ini membahas konsep-konsep inti dalam TensorFlow 2, seperti tf.Variable, tf.Tensor, dan tf.Operation. Dijelaskan bagaimana komponen-komponen dasar ini digunakan untuk membangun model pembelajaran mesin menggunakan TensorFlow, serta pendekatan intuitif yang ditawarkan oleh TensorFlow 2 dibandingkan versi sebelumnya.

### Bab 3: Keras and data retrieval in TensorFlow 2

Bab ini memperkenalkan Keras sebagai API tingkat tinggi dalam TensorFlow untuk membangun model pembelajaran mesin. Dijelaskan bagaimana menggunakan API model-building Keras (Sequential, Functional, dan Sub-classing) dan bagaimana cara mengambil data menggunakan berbagai utilitas TensorFlow seperti API tf.data dan Keras DataGenerators.

### Bab 4: Dipping toes in deep learning

Fokus bab ini adalah pada implementasi model deep learning dasar. Bab ini memperkenalkan jaringan saraf terhubung penuh (fully connected networks), autoencoders, jaringan saraf konvolusional (CNN), dan jaringan saraf rekuren (RNN), dengan penekanan pada pemahaman data yang mendasarinya dan implementasi model.

### Bab 5: State-of-the-art in deep learning: Transformers

Bab ini memperkenalkan model Transformer, yang telah merevolusi pemrosesan bahasa alami (NLP). Bab ini membahas arsitektur jaringan Transformer, mekanisme self-attention, dan multi-head attention. Dijelaskan bagaimana Transformer meningkatkan kinerja dalam tugas NLP seperti penerjemahan mesin dan ringkasan teks.

### Bab 6: Teaching machines to see: Image classification with CNNs

Bab ini membahas jaringan saraf konvolusional (CNN) untuk tugas klasifikasi gambar. Di sini, dijelaskan bagaimana mengimplementasikan CNN, dimulai dengan analisis data gambar dasar, pembuatan pipeline data, dan pelatihan model seperti jaringan Inception untuk klasifikasi gambar yang lebih canggih.

### Bab 7: Teaching machines to see better: Improving CNNs and making them confess

Melanjutkan bab sebelumnya, bab ini membahas teknik untuk meningkatkan kinerja CNN, termasuk arsitektur model yang lebih kompleks dan strategi regularisasi untuk mencegah overfitting.

### Bab 8: Telling things apart: Image segmentation

Bab ini fokus pada segmentasi gambar, yaitu tugas yang mengklasifikasikan setiap piksel dalam gambar ke dalam kategori yang telah ditentukan. Dijelaskan berbagai model segmentasi dan teknik untuk pengenalan objek secara rinci.

### Bab 9: Natural language processing with TensorFlow: Sentiment analysis

Bab ini mengupas tugas analisis sentimen menggunakan TensorFlow. Dijelaskan cara memproses teks, pembersihan data, dan pelatihan model untuk mengklasifikasikan sentimen dari data teks, menunjukkan penerapan TensorFlow dalam tugas NLP.

### Bab 10: Natural language processing with TensorFlow: Language modeling

Bab ini membahas pemodelan bahasa, yang merupakan tugas dasar dalam NLP. Dijelaskan teknik untuk memprediksi probabilitas urutan kata dan memperkenalkan model sequence-to-sequence menggunakan TensorFlow.

### Bab 11: Sequence-to-sequence learning: Part 1

Bab ini memperkenalkan model sequence-to-sequence, yang digunakan untuk tugas seperti penerjemahan mesin, pengenalan ucapan, dan peringkasan. Bagian pertama ini fokus pada arsitektur encoder-decoder dan penerapannya dalam masalah pembelajaran urutan.
